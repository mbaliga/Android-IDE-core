# Content rating (IARC) notes

- Category: utility/productivity tool with **user-directed AI text + image
  generation**.
- The questionnaire asks about AI features (post-2024 IARC revisions): answer
  that the app generates text/images from user prompts, with safety-tuned
  models in the curated catalog; users can import models (user-supplied
  content); outputs can be flagged in-app.
- No gambling, no real-money, no user-to-user communication, no location
  sharing, no ads.
- Expect a Teen-or-higher style rating given free-form generative output;
  answer the violence/sexual-content items per the questionnaire's phrasing
  about *possible user-generated/AI output*, not curated app content.
